# Pushup Form Classifier

A computer vision system that classifies pushup form as **correct** or **incorrect** using MediaPipe pose estimation and temporal neural networks. Given a video of someone doing a pushup, the system extracts body joint positions frame-by-frame and feeds them into a sequence model that learned what good form looks like.

**Dataset:** [Kaggle — Push-up Exercise Dataset](https://www.kaggle.com/datasets/mohamedhanyyy/push-up-exercise-dataset)

## Results

| Model | Accuracy | F1 | Precision | Recall |
|-------|----------|----|-----------|--------|
| Threshold baseline | 0.733 | 0.778 | 0.636 | 1.000 |
| MLP | 0.533 | 0.000 | 0.000 | 0.000 |
| LSTM | 0.800 | 0.800 | 0.750 | 0.857 |
| Transformer | 0.600 | 0.500 | 0.600 | 0.429 |
| **CNN + LSTM** | **0.867** | **0.857** | **0.857** | **0.857** |

CNN+LSTM is the top performer, beating the geometric threshold baseline by ~13 percentage points on accuracy. The threshold baseline uses a single fixed rule: mean back alignment ≥ 160° during the bottom phase → good form.

## Data

The dataset contains **100 pushup video clips** (50 correct form, 50 incorrect form) sourced from Kaggle. Keypoints were pre-extracted using MediaPipe BlazePose, giving 33 body landmarks × (x, y) per frame over 150 frames per clip — a feature vector of shape `(150, 66)` per clip.

Preprocessing applied to all sequences:
- **Translation** — hip midpoint moved to origin
- **Scale normalisation** — divided by torso height (shoulder-to-hip distance)
- **Augmentation (training only)** — horizontal flip, ±20% speed jitter, Gaussian noise

---

## Models

All four models take input `(batch, 150, 66)` and output `(batch, 2)` logits.

| Model | Description |
|-------|-------------|
| **MLP** | Mean-pools all frames → 3-layer MLP. Ignores time entirely. |
| **LSTM** | Bidirectional LSTM over the full sequence. Uses concatenated final hidden states. |
| **CNN + LSTM** | Two 1D conv layers extract local patterns, then bidirectional LSTM reads the result. |
| **Transformer** | Multi-head self-attention with a learnable CLS token for classification. |

Training: AdamW + cosine annealing LR schedule + early stopping (patience = 10).
